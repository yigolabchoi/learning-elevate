## 1. 교사 시나리오

### 시나리오 1-1: 첫 커리큘럼 등록

**사용자**: 김영희 선생님 (중학교 영어 교사, 5년 차)

**상황**: Learning Elevate를 처음 사용하며 자신의 중1 영어 커리큘럼을 등록하려고 함

**플로우**:

```
1. 로그인 후 대시보드 진입
   ↓
2. "커리큘럼 관리" 메뉴 클릭
   ↓
3. "새 커리큘럼 만들기" 버튼 클릭
   ↓
4. 기본 정보 입력
   - 과목: 영어
   - 학년: 중학교 1학년
   - 커리큘럼 이름: "2025 중1 영어 기초"
   ↓
5. 첫 번째 단원 추가
   - 단원명: "현재완료 시제"
   - 학습 목표: "현재완료 시제의 개념을 이해하고 활용할 수 있다"
   - 핵심 개념: [현재완료, have/has, 과거분사, for/since]
   - 예시 문제 추가 (선택)
   ↓
6. 추가 단원 계속 입력 (전치사, 관계대명사 등)
   ↓
7. "저장" 버튼 클릭
   ↓
8. 성공 메시지: "커리큘럼이 저장되었습니다"
   ↓
9. 자동으로 문제 생성 시작
   - "문제를 생성하고 있습니다... (예상 시간: 2분)"
   ↓
10. 문제 생성 완료 알림
    - "현재완료 시제 단원의 문제 10개가 생성되었습니다"
    - [미리보기] 버튼 클릭하여 확인
```

**예상 소요 시간**: 15-20분 (단원 5개 기준)

**기대 효과**:
- ✅ 한 번 등록으로 학기 전체 커리큘럼 관리
- ✅ 자동 문제 생성으로 준비 시간 90% 단축

### 시나리오 1-2: 학생 과제 배정 및 검토

**사용자**: 김영희 선생님

**상황**: 중1-A반 학생들에게 "현재완료 시제" 과제를 배정하고, 제출된 답안을 검토함

**플로우**:

```
1. 대시보드에서 "과제 만들기" 클릭
   ↓
2. 과제 설정
   - 단원: "현재완료 시제"
   - 대상: 중1-A반 (학생 25명)
   - 문제 수: 10개
   - 마감일: 2025-11-20
   - 난이도: 중급
   ↓
3. "과제 배정" 버튼 클릭
   ↓
4. 학생들에게 자동 알림 발송
   ↓
   
[2일 후]

5. 대시보드에서 "검토 대기: 18건" 확인
   ↓
6. "일괄 검토" 클릭
   ↓
7. 학생별 결과 확인
   - 김철수: 5/10 (50%) - 검토 필요
   - 이영희: 9/10 (90%) - 자동 승인 가능
   - 박민수: 7/10 (70%) - 검토 필요
   ↓
8. 김철수 답안 상세 검토
   - Q3 확인: AI 피드백 "현재완료 시제는..."
   - 교사 추가 코멘트 입력: "철수야, 'for'와 'since'의 차이를 다시 복습해보자"
   - 채점 확정
   ↓
9. 모든 학생 검토 완료 후 "최종 컨펌" 클릭
   ↓
10. 학생/학부모에게 결과 공개 알림 발송
```

**예상 소요 시간**: 
- 과제 배정: 2분
- 검토: 학생당 평균 2분 (25명 = 50분) → AI 도움으로 기존 2시간에서 단축

**기대 효과**:
- ✅ 채점 시간 60% 단축
- ✅ 학생별 맞춤 피드백 제공
- ✅ 약점 데이터 자동 수집

### 시나리오 1-3: 학생 약점 분석 및 맞춤 학습 추천

**사용자**: 김영희 선생님

**상황**: 김철수 학생이 현재완료 시제에서 반복적으로 어려움을 겪고 있음을 발견

**플로우**:

```
1. 대시보드에서 "학생 관리" 클릭
   ↓
2. 김철수 학생 선택
   ↓
3. "학습 분석" 탭 확인
   - 취약 영역 TOP 3:
     1. 현재완료 시제 (35% 정답률)
     2. 전치사 (45% 정답률)
     3. 관계대명사 (60% 정답률)
   ↓
4. "현재완료 시제" 클릭하여 상세 분석 확인
   - 최근 3회 시도 모두 오답
   - 주요 실수 패턴: "have/has 누락", "과거분사 형태 오류"
   ↓
5. "맞춤 학습 추천" 버튼 클릭
   ↓
6. AI가 자동 생성한 추천 학습 계획 확인
   - 현재완료 기초 개념 복습 (10문제)
   - 현재완료 vs 과거시제 비교 (10문제)
   - 현재완료 실전 연습 (15문제)
   ↓
7. 추천 학습을 김철수에게 배정
   ↓
8. 김철수 학생에게 알림 발송
   - "선생님이 맞춤 학습을 추천했어요!"
```

**예상 소요 시간**: 5분

**기대 효과**:
- ✅ 학생 약점 정밀 파악
- ✅ 데이터 기반 맞춤 학습 제공
- ✅ 학습 효율 향상

## 2. 학생 시나리오

### 시나리오 2-1: 첫 로그인 및 과제 풀이

**사용자**: 김철수 (중학교 1학년)

**상황**: 선생님이 보낸 초대 링크로 처음 가입하고 첫 과제를 풀어봄

**플로우**:

```
1. 이메일로 받은 초대 링크 클릭
   ↓
2. 회원가입 화면
   - 이름: 김철수
   - 이메일: chulsoo@example.com
   - 비밀번호 설정
   - 학부모 이메일 입력 (선택)
   ↓
3. 가입 완료 후 자동 로그인
   ↓
4. 환영 화면
   - "안녕하세요, 철수님! Learning Elevate에 오신 것을 환영합니다"
   - 간단한 튜토리얼 (스킵 가능)
   ↓
5. 대시보드 진입
   - "오늘의 학습" 섹션에 과제 1개 표시
   - "현재완료 시제 연습" (마감: 2일 후)
   ↓
6. 과제 클릭하여 시작
   ↓
7. 문제 풀이 시작
   - Q1: "I ___ (live) in Seoul for 5 years."
   - 답 입력: "have lived"
   - [제출] 클릭
   ↓
8. 즉시 피드백
   - ✅ "정답입니다!"
   - 다음 문제로 자동 이동
   ↓
9. Q3에서 오답
   - 입력: "lived"
   - ❌ "틀렸습니다"
   - [설명 보기] 클릭
   ↓
10. AI 피드백 확인
    - "현재완료 시제는 'have/has + 과거분사' 형태로..."
    - 정답 예시 및 추가 설명
    - [이해했어요] 클릭
    ↓
11. 모든 문제 완료 후 결과 화면
    - "5/10 정답 (50%)"
    - "조금 더 연습이 필요해요!"
    - [오답 다시 보기] [추가 연습 문제]
```

**예상 소요 시간**: 15-20분

**학습 경험**:
- ✅ 즉각적인 피드백으로 학습 효과 향상
- ✅ 자세한 설명으로 개념 이해
- ✅ 게임처럼 재미있는 학습 경험

### 시나리오 2-2: 추천 학습 및 약점 보완

**사용자**: 김철수

**상황**: 과제를 마친 후, AI가 추천하는 맞춤 학습을 진행함

**플로우**:

```
1. 대시보드에서 "추천 학습" 탭 클릭
   ↓
2. 맞춤 추천 목록 확인
   - 🎯 "현재완료 시제 집중 연습" (우선 추천)
   - 📚 "전치사 기초 다지기"
   - 🔄 "지난주 복습: 관계대명사"
   ↓
3. "현재완료 시제 집중 연습" 클릭
   ↓
4. 학습 정보 확인
   - 예상 소요 시간: 15분
   - 문제 수: 10개
   - 난이도: 기초
   - "이 학습은 철수님의 약점을 보완하기 위해 추천되었어요"
   ↓
5. [시작하기] 클릭
   ↓
6. 문제 풀이 (이전보다 쉬운 난이도)
   - Q1: 기본 개념 확인 문제

   - Q6-10: 실전 문제
   ↓
7. 결과 확인
   - "8/10 정답 (80%)"

   - 레벨업 알림: "Level 4 → Level 5"
   ↓
8. 성취 배지 획득 (향후 기능)
   - 🏆 "현재완료 마스터 브론즈"
```

**예상 소요 시간**: 15분

**학습 효과**:
- ✅ 약점 집중 보완

```
   - "철수야, 기본 시제는 정말 잘하고 있구나!"
   ↓
3. 약점 함께 확인
   - "현재완료 시제가 조금 어려운가 보네?"
   - 자녀: "네, 개념이 헷갈려요"
   ↓
4. 함께 목표 설정
   - "이번 주는 현재완료 시제 집중 연습하자"
   - "하루 20분씩, 주 3회 학습하기"
   ↓
5. 학부모 앱에서 "학습 목표 설정" (향후 기능)
   - 목표: 현재완료 시제 정답률 70% 달성
   - 기간: 2주
   - 알림 설정: 매일 저녁 7시
   ↓
6. 다음 주 리포트에서 진행 상황 확인 예정
```

**효과**:
- ✅ 부모-자녀 소통 강화
- ✅ 목표 지향적 학습
- ✅ 가정에서의 학습 지원

## 4. 통합 시나리오: 한 학생의 학습 여정

### 배경
- **학생**: 김철수 (중1)
- **교사**: 김영희 선생님
- **학부모**: 김철수 어머니
- **기간**: 4주

### Week 1: 시작

```
[교사]
- 커리큘럼 등록 (현재완료 시제 포함)
- 중1-A반에 첫 과제 배정

[학생]
- 첫 로그인 및 과제 풀이
- 결과: 5/10 (50%)
- 현재완료 시제 개념 부족 확인

[학부모]
- 첫 리포트 수신
- 약점 영역 확인
- 자녀와 학습 목표 설정
```

### Week 2: 집중 학습

```
[교사]
- 김철수의 약점 분석 확인
- 맞춤 학습 추천 (현재완료 집중)

[학생]
- AI 추천 학습 3회 완료
- 정답률 50% → 65% 향상
- Level 4 → Level 5 레벨업

[학부모]
- 주간 리포트에서 향상 확인
- 자녀 칭찬 및 격려
```

### Week 3: 실력 향상

```
[교사]
- 현재완료 심화 과제 배정
- 김철수 답안 검토 시 긍정적 코멘트

[학생]
- 과제 결과: 8/10 (80%)
- 성취 배지 획득
- 자신감 향상

[학부모]
- 리포트에서 큰 향상 확인
- 선생님 칭찬 코멘트 확인
- 자녀와 함께 기뻐함
```

### Week 4: 마스터

```
[교사]
- 김철수를 다른 학생들의 멘토로 지정 (향후 기능)
- 다음 단원(전치사) 준비

[학생]
- 현재완료 시제 마스터 (정답률 90%)
- 다른 학생 도와주기
- 학습 동기 최고조

[학부모]
- 한 달 종합 리포트 수신
- 자녀의 성장에 만족
- 서비스 추천 의향 높음
```
