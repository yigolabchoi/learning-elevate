## 서비스 핵심 목적

### 문제점
```
❌ 한 명의 선생님이 수십 명의 학생을 담당
❌ 반복적인 채점 및 피드백 작성에 시간 소모
❌ 개별 학생 상태 파악이 어려움
❌ 우선순위 판단이 어려워 효율적 시간 배분 불가
❌ 같은 코멘트를 반복적으로 작성
```

### 해결 방안
```
✅ AI가 자동으로 채점 및 상세 피드백 생성
✅ AI가 주의가 필요한 학생 자동 감지
✅ AI가 업무 우선순위를 자동 정렬
✅ 한 번의 클릭으로 여러 제출물 승인
✅ 자주 사용하는 코멘트를 템플릿으로 저장
✅ AI가 학생별 맞춤 과제 자동 추천
```

## AI 자동화 기능 개요

### 1. 🤖 AI 인사이트 대시보드
**위치:** `/dashboard` (Teacher 로그인 시 최상단)

**기능:**
- AI가 자동으로 오늘 해야 할 일을 우선순위별로 정렬
- 예상 소요 시간 자동 계산
- 주의가 필요한 학생 자동 감지
- AI 신뢰도 높은 제출물 자동 필터링
- 학생별 맞춤 과제 자동 추천

**업무 절감:**
- Before: 30분 (모든 제출물 확인, 학생 상태 파악, 과제 계획)
- After: 5분 (AI가 정리한 요약만 확인)
- **83% 시간 절감**

---

### 2. ⚡ 빠른 승인 시스템
**위치:** AI 인사이트 대시보드 또는 `/teacher/submissions`

**기능:**
```typescript
// AI 신뢰도 95% 이상인 제출물만 표시
- AI 채점: 95점
- AI 신뢰도: 98%
- 학생: 우수 학생 (평균 90점 이상)
→ [일괄 승인] 버튼 한 번에 모두 승인
```

**특징:**
- AI가 높은 신뢰도로 채점한 제출물만 자동 필터링
- 클릭 한 번으로 여러 제출물 동시 승인
- 학생에게 즉시 피드백 전달

**업무 절감:**
- Before: 제출물 12개 × 2분 = 24분
- After: 일괄 승인 1번 클릭 = 10초
- **99.3% 시간 절감**

---

### 3. 🎯 AI 추천 과제 시스템
**위치:** AI 인사이트 대시보드

**동작 방식:**
1. AI가 학생들의 학습 데이터 분석
   - 최근 제출물 점수
   - 개념별 정확도
   - 자주 틀리는 유형
   - 학습 진도

2. 약점이 비슷한 학생 자동 그룹화
   ```
   예: Grammar 약한 학생 3명 감지
   → "Grammar Fundamentals Review" 과제 자동 추천
   → [배정] 버튼 클릭 한 번으로 3명에게 동시 배정
   ```

3. 과제 내용도 AI가 자동 생성
   - 학생 수준에 맞는 난이도
   - 약점 개념 집중 문제
   - 적절한 문제 수

**업무 절감:**
- Before: 과제 선정 10분 + 개별 배정 5분 = 15분
- After: AI 추천 확인 1분 + 클릭 1번 = 1분
- **93% 시간 절감**

---

### 4. 💬 템플릿 코멘트 시스템
**위치:** 제출물 검토 시 사이드바

**기능:**
```typescript
// 자주 사용하는 코멘트 저장
템플릿 1: "잘했어요!"
템플릿 2: "문법 오류 지적"
템플릿 3: "어휘 향상 필요"

// 클릭 한 번에 코멘트 적용
→ 클릭 → 자동 복사 → 붙여넣기
```

**카테고리:**
- 칭찬 (praise)
- 개선 필요 (improvement)
- 문법 (grammar)
- 어휘 (vocabulary)
- 일반 (general)

**통계:**
- 각 템플릿 사용 횟수 자동 기록
- 가장 많이 사용하는 템플릿 상단 정렬

**업무 절감:**
- Before: 코멘트 작성 3분 × 10명 = 30분
- After: 템플릿 선택 10초 × 10명 = 1.7분
- **94% 시간 절감**

---

### 5. ⚠️ 주의 필요 학생 자동 감지
**위치:** AI 인사이트 대시보드 최상단 (빨간색 경고)

**AI 감지 기준:**
```typescript
// 다음 조건 중 하나라도 해당하면 알림
1. 최근 3주간 성적이 지속 하락
2. 평균 점수가 70점 미만
3. 출석률이 85% 미만
4. 과제 제출률이 70% 미만
5. 5일 이상 플랫폼 미접속
6. 특정 개념에서 정확도 50% 미만
```

**표시 정보:**
```
⚠️ Jung Woo-jin
- 현재 점수: 65점
- 비활동: 5일
- 이유: "최근 3주간 성적이 지속적으로 하락"
→ [확인] 버튼 → 학생 상세 페이지로 즉시 이동
```

**자동 액션 제안:**
- 일대일 상담 권장
- 보충 과제 추천
- 학부모 알림 발송 제안

**업무 절감:**
- Before: 학생별 상태 수동 확인 20분
- After: AI가 자동 감지 및 정렬 = 0분
- **100% 시간 절감** (자동화)

### 6. 📊 학생 그룹 자동 분류
**위치:** `/teacher/students`

**AI 자동 분류:**
```typescript
// 실시간 필터링
1. 우수 학생 (90점 이상)
2. 양호 학생 (70-89점)
3. 주의 필요 (70점 미만)
4. 최근 활동 (3일 이내)
5. 비활동 (7일 이상)
```

**빠른 액션:**
- 그룹별 일괄 과제 배정
- 그룹별 메시지 발송
- 그룹별 리포트 생성

### 7. 🚀 문제 자동 생성
**위치:** `/teacher/curricula/new`

**기능:**
```
1. 선생님이 PDF/이미지 업로드
   (예: 작년 중간고사 문제)

2. AI가 자동 분석 (1.5초)
   - 문제 유형 파악
   - 난이도 판단
   - 핵심 개념 추출

3. AI가 유사 문제 자동 생성 (3초)
   - 같은 유형, 다른 문제
   - 같은 난이도
   - 같은 개념

4. 선생님이 확인 후 승인
   → 즉시 과제로 배정 가능
```

**업무 절감:**
- Before: 문제 직접 제작 30분
- After: 파일 업로드 + AI 생성 = 2분
- **93% 시간 절감**

### 8. 🎓 AI 자동 채점 및 피드백
**위치:** 학생 제출 시 자동 실행

**AI가 자동으로:**
```typescript
1. 채점 (0-100점)
2. 정답/오답 판별
3. 상세 피드백 생성
   - 강점 분석
   - 약점 분석
   - 학습 제안
   - 개선 방법
4. 개념별 정확도 계산
5. 종합 리포트 생성
```

**선생님은:**
```typescript
// AI 결과를 검토만 하면 됨
1. AI 피드백 확인 (10초)
2. 점수 조정 (필요시)
3. 추가 코멘트 작성 (선택)
4. 승인 버튼 클릭

// 또는
1. "빠른 승인" 버튼 한 번 클릭 (AI 신뢰도 높은 경우)
```

**업무 절감:**
- Before: 수동 채점 + 피드백 작성 = 5분/명
- After: AI 검토 + 승인 = 30초/명
- **90% 시간 절감**
