## 1. 화면 구조 개요

### 1-1. 전체 IA (Information Architecture)

```
Learning Elevate
│
├── 인증 (Authentication)
│   ├── 로그인
│   ├── 회원가입
│   └── 비밀번호 재설정
│
├── 교사 (Teacher)
│   ├── 대시보드
│   ├── 커리큘럼 관리
│   │   ├── 커리큘럼 목록
│   │   ├── 커리큘럼 등록/편집
│   │   └── 단원 상세
│   ├── 문제 세트 관리
│   │   ├── 문제 세트 목록
│   │   ├── 문제 생성
│   │   └── 문제 미리보기
│   ├── 과제 관리
│   │   ├── 과제 목록
│   │   ├── 과제 생성
│   │   └── 과제 배정
│   ├── 학생 관리
│   │   ├── 학생 목록
│   │   ├── 학생 상세
│   │   └── 학습 분석
│   └── 검토 및 컨펌
│       ├── 검토 대기 목록
│       └── 답안 검토
│
├── 학생 (Student)
│   ├── 대시보드
│   ├── 오늘의 학습
│   ├── 추천 학습
│   ├── 문제 풀이
│   │   ├── 문제 화면
│   │   └── 결과 화면
│   └── 학습 기록
│       ├── 전체 통계
│       └── 상세 분석
│
├── 학부모 (Parent)
│   ├── 대시보드
│   ├── 자녀 선택
│   ├── 학습 리포트
│   │   ├── 요약
│   │   └── 상세 분석
│   └── 설정
│
└── 공통
    ├── 프로필
    ├── 설정
    └── 알림
```
